1. **Create Database:**
	- CREATE DATABASE dbName

2. **Get all databases:**
	- forward-slash + l;

3. **Quite from Database:**
	- qi;

4. **Redirect to Database:**
	- forward-slash + c dbName;

5. **Get specific table in a Database:**
	- forward-slash + dt;

6. **Delete a table:**
	- DROP TABLE tableName;

7. **Get data of a specific table:**
	- SELECT * FROM tableName;
